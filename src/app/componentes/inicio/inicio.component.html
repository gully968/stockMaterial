InventarioAPP

Bienvenido al sistema de inventario. Presione sobre Acceder para realizar la validación de usuario y poder tener acceso total a las funciones del sistema. Si ya se encuentra logueado debe visualizar un menú sobre la barra principal con las funciones disponibles.

  • Tablas: Utilice esta opción para acceder a las diferentes tablas del sistema y completelas con los datos que requiera

  • Informes: Esta opción proporciona los datos ingresados para ser consultados por pantalla o impresos

  • Movimientos: Desde aquí puede acceder a ingresar y retirar productos del inventario, pudiendo imprimir un ticket
